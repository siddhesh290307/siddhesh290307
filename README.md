## 🚀 Some Things I've Built

**Cerebra — RAG Research Assistant** · `FastAPI · React · Groq · Sentence Transformers`  
End-to-end RAG pipeline for searching academic papers. Pulls from arXiv, chunks PDFs, embeds them, and lets you query across multiple papers. React frontend, JWT auth, deployed on Vercel.

**Neural Machine Translation** · `TensorFlow · Seq2Seq · Bahdanau Attention · Beam Search`  
English → French translation. Got a 93% BLEU improvement over baseline by implementing attention and beam search from scratch.

**ResNet-34 from Scratch** · `TensorFlow · CIFAR-10`  
Implemented residual blocks and skip connections manually. Landed at **85.27% test accuracy**.

**Food Classification** · `EfficientNetV2B0 vs MobileNetV2 · Streamlit`  
Fine-tuned two architectures on a 20-class Indian food dataset and deployed the better one as a Streamlit app.
