### 🌍 Neural Machine Translation — Attention vs Beam Search
> *Python · TensorFlow · Seq2Seq · LSTMs · Bahdanau Attention*

Comparative study of decoding strategies in English → French translation.

| Strategy | BLEU Score |
|---|---|
| Vanilla LSTM | 0.14 |
| Attention + Greedy Search | ~0.21 |
| **Attention + Beam Search (k=3)** | **0.27** |

- Implemented **Bahdanau Additive Attention** to solve the fixed-length vector bottleneck
- 93% improvement in BLEU score over the baseline vanilla LSTM
